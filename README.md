# yolo_annotations_heatmap
Heatmap from YOLOv5 annotations. Works only with images of similar shape

dependencies: matplotlib, numpy

## How to use
```
import yolov5_heatmap as yhp

yhp.show_heatmap('train/labels', 640, 640)
```
